## Contact

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/lhlRahman/Habibs-Portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the server:
   ```bash
   npm start
   ```
